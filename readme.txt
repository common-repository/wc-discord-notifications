Notifications on Discord for Woocommerce will send a notification to your chosen Discord channel, notifying you of a new order or order status change.

== Description ==

= Send notifications from your Woocommerce store to your chosen Discord channel. =

Notifications on Discord for Woocommerce allows you to receive discord notifications for each Woocommerce order status.

== Installation ==

= Download, Install and Activate! =
1. Go to Plugins > Add New to find and install Notifications on Discord for Woocommerce.
2. Or, download the latest version of the plugin, then go to Plugins > Add New and click the "Upload Plugin" button to upload your .zip file.
3. Activate the plugin.

= Complete the Initial Plugin Setup =

1. Visit [Intro to Webhooks](https://support.discord.com/hc/en-us/articles/228383668-Intro-to-Webhooks) to create a webhook in Discord.
2. Copy the webhook and login to your WordPress dashboard. 
3. Go to Settings > Discord Notifications and enable the status you'd like to be notified for followed by entering in your Webhook URL.

== Frequently Asked Questions == 

= I'm not receiving any discord notifications on my channel =
Discord won't accept notifications from an offline site. Be sure to send your discord notifications from an online site that is publicly accessible.
